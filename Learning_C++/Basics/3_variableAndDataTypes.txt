| **Category**             | **Description**            | **Examples**                                |
| ------------------------ | -------------------------- | ------------------------------------------- |
| **1. Basic / Primitive** | Fundamental built-in types | `int`, `float`, `char`, `double`, `bool`    |
| **2. Derived**           | Derived from basic types   | `array`, `pointer`, `function`, `reference` |
| **3. User-defined**      | Created by programmers     | `struct`, `class`, `union`, `enum`          |


| **Data Type** | **Meaning**                        | **Size (approx.)** | **Example**                            |
| ------------- | ---------------------------------- | ------------------ | -------------------------------------- |
| `int`         | Integer numbers (no decimal)       | 4 bytes            | `int age = 20;`                        |
| `float`       | Decimal numbers (single precision) | 4 bytes            | `float pi = 3.14;`                     |
| `double`      | Decimal numbers (double precision) | 8 bytes            | `double g = 9.81;`                     |
| `char`        | Single character                   | 1 byte             | `char grade = 'A';`                    |
| `bool`        | Boolean value (`true`/`false`)     | 1 byte             | `bool passed = true;`                  |
| `void`        | No value / empty                   | 0 bytes            | Used for functions that return nothing |


| **Type**      | **Description**                   | **Example**             |
| ------------- | --------------------------------- | ----------------------- |
| **Array**     | Collection of same data type      | `int marks[5];`         |
| **Pointer**   | Stores memory address             | `int *ptr;`             |
| **Function**  | A block of code returning a value | `int sum(int a, int b)` |
| **Reference** | Alias name for a variable         | `int &ref = a;`         |


| **Type**                 | **Description**                        | **Example**                                |
| ------------------------ | -------------------------------------- | ------------------------------------------ |
| **Structure (`struct`)** | Group of different data types          | `struct Student { int id; string name; };` |
| **Class (`class`)**      | Like a struct but with OOP features    | `class Car { ... };`                       |
| **Union (`union`)**      | Shares memory for all members          | `union Data { int i; float f; };`          |
| **Enum (`enum`)**        | Defines set of named integer constants | `enum Day {Sun, Mon, Tue};`                |


| **Modifier** | **Used With** | **Example**           |
| ------------ | ------------- | --------------------- |
| `short`      | int           | `short int x;`        |
| `long`       | int, double   | `long double pi;`     |
| `signed`     | int, char     | `signed char c;`      |
| `unsigned`   | int, char     | `unsigned int count;` |
